## 3. Watcher 機能の UI 表現方針

| 通知重要度 🟠 | 🟠       | High               |
| 通知重要度 🟡 | 🟡       | Normal             |

## 6. ユーザー調査対象

### 6.1 ターゲットユーザー

| ユーザー層         | 比率 | 優先事項                 |
| :----------------- | :--- | :----------------------- |
| CEO/管理者         | 30%  | 即時通知、重要度判断     |
| チームリーダー     | 40%  | タスク監視、進捗把握     |
| エージェント管理者 | 20%  | 複数エージェント状態確認 |
| 一般ユーザー       | 10%  | シンプルな意思決定UI     |

### 6.2 ユースケース

1. **外出中の意思決定**: 移動中に重要な承認を行う
2. **タスク監視**: 長時間タスクの進捗を通知で把握
3. **緊急対応**: タイムアウトタスクの即時再開
