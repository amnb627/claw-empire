## 3. 文字化け時のフォールバックUI仕様

### 3.1 重大度別表示ルール

| 重大度 | UI表現          | アイコン | アクション     |
| :----- | :-------------- | :------- | :------------- |
| **P0** | `[ENCODE_ERR]`  | ⚠️       | 再試行ボタン   |
| **P1** | `??????????`    | ⚠️       | バグ報告リンク |
| **P2** | `日本語テスト?` | ℹ️       | 表示           |
| **P3** | 絵文字代替      | 📝       | 表示           |

### 3.2 トースト通知仕様

```
┌─────────────────────────────────┐
│ ⚠️ 文字の表示に問題が発生しました │
│                                 │
│ 正しく表示されない場合は、       │
│ [バグを報告] してください       │
│                    [閉じる]      │
└─────────────────────────────────┘
```

### 3.3 豆腐文字代替表示

```
┌─────────────────────────────────┐
│ 📝 テキスト                     │
│                                 │
│ この環境では絵文字が正しく       │
│ 表示されない可能性があります     │
│                                 │
│ [絵文字を有効にする]            │
└─────────────────────────────────┘
```

## 5. 検証手順

### 5.1 スクリーンショット撮影

1. 各デバイス/ブラウザでテストページを開く
2. テスト文字列を表示
3. スクリーンショットを撮影
4. 結果をスプレッドシートに記録

### 5.2 回帰テスト

- 新しいフォントバージョンリリース時
- iOS/Android メジャーアップデート時
- ブラウザメジャーアップデート時
