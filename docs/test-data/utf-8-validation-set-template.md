## 3. エンコーディング検証パターン

### 3.1 UTF-8正常パターン

| 入力 | バイト列(hex) | 期待結果 |
| :--- | :------------ | :------- |
| 日   | E6 97 A5      | 日       |
| 本   | E6 9C AC      | 本       |
| 語   | E8 AA 9E      | 語       |

### 3.2 文字化けパターン（検証用）

| パターン      | 現象      | 原因                       |
| :------------ | :-------- | :------------------------- |
| Shift-JIS誤認 | ˇÌßÞÝØ    | UTF-8をShift-JISとして解釈 |
| EUC-JP誤認    | ƒÉŒÒ      | UTF-8をEUC-JPとして解釈    |
| Latin-1誤認   | æ—¥æœ¬èªž | UTF-8をLatin-1として解釈   |
